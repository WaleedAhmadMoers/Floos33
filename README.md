# floos33

B2B stocklot marketplace connecting EU and MENA businesses.

## Current status

- Accounts app
  - signup
  - login
  - password reset
  - dashboard
  - profile/settings
  - buyer verification
  - seller verification request

- Companies app
  - create company
  - company profile
  - company edit

- Stocklots app
  - categories
  - subcategories
  - listing create/edit
  - marketplace preview

## Tech stack

- Django
- SQLite (development)

## Notes

This project is being built step by step, app by app, starting with a stocklot-focused B2B MVP.
